# Discord Quest Spoofer

A simple userscript that automatically spoofs Discord Quests by faking your progress.

## What Does It Do?

This script tricks Discord into thinking you have completed quests (watching videos, playing games, streaming, or voice activity) much faster than normal. It injects fake progress directly into Discord.

## Features
- Auto detects active quests
- Supports: WATCH_VIDEO, PLAY_ON_DESKTOP, STREAM_ON_DESKTOP, PLAY_ACTIVITY
- Fast quest completion
- Works on Discord Web & Desktop


---


| Quest Type              | Platform     | Description                          |
|-------------------------|--------------|--------------------------------------|
| WATCH_VIDEO             | Web/Mobile   | Fake video watching progress         |
| PLAY_ON_DESKTOP         | Desktop      | Fake game playing time               |
| STREAM_ON_DESKTOP       | Desktop      | Fake streaming time                  |
| PLAY_ACTIVITY           | Desktop/Web  | Fake voice call / activity time      |

## How to Use
1. Open Discord
2. Press `Ctrl + Shift + I`
3. Go to Console
4. Paste the script and press Enter
